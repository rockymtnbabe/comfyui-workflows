## Qwen Edit Usage Patterns

### Style / Clothing Change
- Image2: ON
- ControlNet: Bypassed
- Result: Clothing and style changes work without pose enforcement

### Pose Transfer
- Image2: ON
- ControlNet: ON
- Result: Pose or structure influences generation

### Minimal Edit
- Image2: OFF
- ControlNet: OFF
- Result: Text-only edits on base image
