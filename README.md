# comfyui-workflows

Workflow library for ComfyUI experiments and production setups.  
This repository contains ComfyUI workflow JSONs for image creation, editing and experimentation.

## Structure
- `workflows/` – exported ComfyUI workflow JSONs
- `notes/` – short notes per workflow/model
- `assets/` – reference images and screenshots (optional)

## Naming convention
`[topic]_[purpose]_[version]_[key_feature]_v#`

Example: `qwen_edit_2511_controlnet_toggle_v1.json`
